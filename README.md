# 📄 About the Project

This is a **landing page for an architectural studio**, created as an educational exercise.

The main goal of the project was to practice:
- semantic HTML structure
- responsive layout building
- working with modern CSS layout systems

---

# 🛠️ Tech Stack

| Technology | Description |
|---|---|
| HTML5 | Semantic markup and page structure |
| CSS3 | Flexbox and Grid layout systems |

---

# ✨ Features

- 🧱 Clean and structured layout
- 📱 Responsive design for different screen sizes
- 🧭 Well-organized sections (hero, services, gallery, contacts)
- 🎯 Grid-based content structure
- 🎨 Flexbox alignment for components
- ⚡ Lightweight static page (no frameworks)
